PROJECT 4 • PYTHON HR ANALYTICS

Employee Performance & Workforce Insights Analysis

This project focuses on analysing employee performance and workforce data using Python. It demonstrates how a data analyst explores, cleans, analyses, and visualises workforce data to uncover meaningful insights that can support business decision-making.

Python Pandas Matplotlib Seaborn HR Dataset Data Cleaning Visual Analytics

📌 Introduction

This project uses a realistic Human Resources dataset that contains employee-level information such as salary, performance scores, attendance rates, departments, job roles, education, and join dates. The dataset is intentionally noisy, containing missing values, duplicates, and inconsistent text formatting to reflect real-world data conditions.

What the Project Is

A Python-based workforce analytics project that demonstrates data loading, cleaning, summarisation, grouping, visualisation, and insight generation.

What Dataset Is Used

A realistic HR dataset containing employee information such as salary, performance, attendance, department, job role, education, and join dates.

What the Goal Is

To clean the dataset, analyse workforce patterns, create visualisations, and generate insights that can support business decision-making.

🎯 Project Objectives

  • Load and inspect the HR dataset in Python.
  • Identify missing values, duplicates, and inconsistent text formatting.
  • Clean and standardise the dataset for reliable analysis.
  • Generate descriptive statistics for key numeric variables.
  • Perform grouping and aggregation by department and gender.
  • Create visualisations that reveal workforce patterns and trends.
  • Summarise findings to support evidence-based decision-making.

4️⃣ Data Cleaning

This section addresses key data quality issues: missing values, duplicates, and inconsistent text formatting. These steps improve reliability and prepare the dataset for meaningful analysis.

Cleaning
Missing Values Before Cleaning
Missing values were identified before cleaning. The output shows that PerformanceScore and AttendanceRate contained null values.
Cleaning
Text Cleaning and Standardisation
Department values were cleaned using string methods to remove extra spaces and convert entries into title case, ensuring consistent categorical values.
Cleaning
Handling Missing Values
Missing performance scores were replaced using the column mean. This preserves the dataset size while supporting continued analysis of workforce performance patterns.
Cleaning
Duplicate Removal
Duplicate records were removed from the dataset. After cleaning, the row count reduced from 370 to 350, indicating that duplicate entries were present.
Cleaning
Cleaned Dataset Check
The post-cleaning structure confirms that the final dataset contains 350 non-null rows across all columns, showing that missing values and duplicates were resolved successfully.
